Kembali berotasi ke blockchain dan smart contract.

Kini berlajut ke A Deep Dive into Ethereum Gas Fees[^1].

Untuk melakukan interaksi di Ethereum, seperti mengirim transaksi atau berinteraksi dengan smart contract diperlukan sumber daya komputasi (*computational resources*). Untuk itu user diperlukan untuk membayar fee. Nah fee ini disebut dengan *gas*.

Ada setidaknya 2 alasan mengapa *gas* diperlukan.

- Kompensasi untuk validator
- Untuk mencegah spam

Evolusi *gas*.

Type 0 - Legacy Mode

Type 2 - EIP-1559
