Dilanjutkan mencoba menjalankan blockchain secara lokal. Menggunakan `anvil`. Pada contoh juga dikenalkan ganache, namun karena pengembangan tidak dilanjutkan sebaiknya menggunakan anvil saja.

Deploy smart contract menggunakan `forge create <NamaContract> --rpc-url <url> --interactive`.

Itu digunakan sebagai contoh. Dan juga Patrick menjelaskan mengenai jangan pernah simpan private key yang ada uangnya di dalam bentuk plain text (yang mana saya rasa saya sangat sering melakukan hal ini).

Dengan janji ini.

```text
I promise to never use my private key associated with real money in plain text.
```

Mungkin kedepannya, saya akan mencari tahu cara amannya dibanding simpan dalam bentuk plain text.
